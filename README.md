# 🌾 GraminKrishi

GraminKrishi is a smart agriculture platform connecting landowners and agricultural workers across Bangladesh.

## Vision

To make it easy for landowners to find workers and for workers to find agricultural jobs through a simple digital platform.

## User Types

- Land Owner
- Agricultural Worker

## Core Features

- User Registration & Login
- Dual Role Account (Owner + Worker)
- Job Posting
- Worker Availability Posts
- Direct Phone Call
- Search & Filter
- Notifications
- Profile Management
- Reviews & Ratings
- Location Based Search
- Admin Dashboard
